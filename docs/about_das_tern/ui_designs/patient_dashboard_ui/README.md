## Medication Card Component

Each medication displays as a card:

| Element | Description |
|---------|-------------|
| **Pill Image** | Actual medication image/icon |
| **Medicine Name** | e.g., Saridon, Paracetamol |
| **Dosage** | e.g., 500mg, 250mg |
| **Quantity** | e.g., 1 tablet, 2 capsules |
| **Scheduled Time** | e.g., 8:00 AM |
| **Action Button** | Blue "á”á¶á“á‘á‘á½á›" (Taken) button |

### Card States

| State | Visual | Button |
|-------|--------|--------|
| **Upcoming** | Gray border | Disabled |
| **Due Now** | Blue highlight | Active blue |
| **Taken** | Green check | Completed |
| **Missed** | Red border | "Missed" label |

## User Stories

- As a patient, I see my medications organized by time of day
- As a patient, I can quickly mark medications as taken
- As a patient, I see my adherence history on the calendar
- As a patient, I can still track medications when offline

---

## Acceptance Criteria

- [ ] Medications grouped by Morning/Afternoon/Night
- [ ] Each section has distinct color theme
- [ ] Medication cards show image, name, dosage, time
- [ ] Quick "Taken" button for each medication
- [ ] Monthly calendar shows adherence status
- [ ] Offline indicator visible when disconnected
