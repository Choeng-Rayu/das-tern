## User Stories

- As a user, I can log in with my phone or email
- As a user, I can see/hide my password
- As a user, I can recover my password if forgotten
- As a user, I can navigate to registration if I don't have an account

---

## Acceptance Criteria

- [ ] Dark blue background theme applied
- [ ] Phone/Email input field functional
- [ ] Password field with show/hide toggle
- [ ] Login button triggers authentication
- [ ] Forgot password link navigates to recovery
- [ ] Registration link navigates to sign up
- [ ] Khmer language labels displayed
