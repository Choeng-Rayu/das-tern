# Theme Directory

## Role
This directory handles the visual styling, theme definitions, and shared theme-related widgets.

## Rules
- **Parent Rules**: Adhere to the rules defined in [../README.md](../README.md).
- **Naming Convention**: Any new widget file created in this directory MUST follow the pattern `widgetName_widget.dart`.

## Implementation Guidelines
- **New Reusable Widget**:
  - Create a new file following the naming convention `widgetName_widget.dart`.
  - Example: `button_widget.dart`, `card_widget.dart`.
